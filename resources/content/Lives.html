Welcome to LIVES game.

In this chapter, we will explain how this game works, so that we will facilitate you the understanding of the code later.

To begin the game, press the ball. The purpose of the game is to obtain a bigger score (indicated top-left) than your friends xD. Everytime you click on the ball, the score increase, but together with the speed of the ball. However, beaware, if you click outside of the ball, you loose one life.

You can have a try now. Good luck!

The CODE

Let's start with the "void setup()". We start by creating the window of the game : we give it a size of 400x400 px. We also give deatils about the ball that it is moving : at first it will be positioned in the middle of the window. We also set the main colour as blue and the set the size of the text from the corners.

The draw function is used to make the ball move to left or right. We also want to make sure that if the ball hits the edges, it will go the opposite way. If we don't do that, the ball would disappear!. We also display the score and the number of lives. Moreover, we need to change the display if the user lost the game : we want to make sure the user KNOWS he lost. As a result, we tell him that if he wanted to start again, he needs to "click to restart",

The "void mousPressed()" function is probably the most important one : it makes sure that when we press on the ball or outside of it, we update the score as well as the speed of the ball. As a result if we hit the target, the speed increases, if not, it decreases and the number of lives available lowers. We also need to make sure that if the numbr of lives is 0, the game would stop. If we did not do that, the game would never stop. As a result, when the game is over, we reset everything to how it was.

Welcome to BASIC STARS application.

Hi. This will be the first application that you will run in Processing. Well, it's not exactly an application because it doesn't really do anything, but, we will present you this one to get you accomodated with the basic main functions that define your application in processing (setup and draw).

void setup() { ... }


This is where you put all the "details" of your application that will remain unchanged. For example: the background color, or the size of the window that your application will run on. More details on this: here.
If you pressed the link above, you will realise you have been redirected to the Processing web-site from which you have previously downloaded your application. And this is the best place from which you can gather a lot of useful information about everything that Processing you can do. So, please keep in mind that it's a very good idea to come back here from time to time and discover some new awsome features you can add to your code.

void draw()


Here is the place where the magic happens. These represents the code of your application. It basically repeats the code inside this function until the program is finished. You can read more details on these here here.

With all these being said, it's time to fill these functions with some code, right? So, we will firstly decide which will be the seize of our window, and in our case, it will be 640 by 360, and you set this by the following line of code being inserted in the void setup() function: size(640, 360);

Next, we will try to draw two simple stars. For this, we will need to make our own function to draw a star, obtiously called star. So, under the draw() function, define our function: void star(float x, float y). Inside the body (the lines between the two curly braces { } ), first type the beginShape(); and endShape(CLOSE); lines of code, one after another. These functions allow us to draw complex shapes that are defined between these two lines.

Now that we used the previous two functions, we can define how our star shape should look like. For this, we will use the vertex() function. All shapes are made out of linked vertices, from a point to another. So, as you can already imagine, we are going to define a series of vertex() functions that represent the points that are going to be linked through vertices, so we can finally a concrete shape. As you can also see from the code that sits under our representation of this basic application. there three more functions: fill(102);,stroke(255); and strokeWeight(2);. The numbers between the braces represent the code colors used in Processing. You can google for these codes, or, Processing also has a color selector to aid in choosing colors. Access this via Tools (from the menu bar) â†’ Color Selector.

fill() sets the color that our shape will have.

stroke() sets the color of the stroke around our shape.

strokWeight() sets the "dimension" of our stroke in pixels.

All these being said, we consider that you can take the code below, paste it into your blank processing page, run it (you should see the same window as below) and then start playing with the function, or even with the shape.

Random Moving Balls

This examples ilustrates how the processing language can be used for expresing ideas that are rather more artistic.

The aplication works in the following way: each time you click on the canvas a new ball will spwan and it move in random directions. The path of each ball is recorded and drawn, creating a aesthetically interesting visual pattern.

Enjoy!

Now, let's take a look at the code!

The setup() is very simple this time, we only need to set our canvas size to 600 by 600 pixels.
For this application we designed a Particle class to define each particle or ball we want to draw.
So, an object(a ball) of type Particle will have x and y coordinates, an array to store
each movement of the ball from the moment of its creation and a method called Update() that does exactly
what the name suggests, chooseing at random where a ball should move next, and adding that ifnormation into our array.
Outside the class, we need a to implement a method that draws the balls we have already created and their specific
prior movemnets. The show() method does this essentially endlessly, being called in the the draw
function, that, as you already know, is executed in an endless loop. The mousePressed() method creates
a new particle object every time the mouse is presed, int the same location as the cursor.
In the draw() method we call the Update() and show() on each particle object
we have created and that is it!
